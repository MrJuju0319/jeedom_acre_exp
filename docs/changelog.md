# Changelog

## 1.1.0
- Réécriture complète du plugin : chaque équipement gère sa propre connexion et son environnement Python virtuel.
- Simplification du fonctionnement : suppression du démon système et utilisation d'un cron Jeedom natif.
- Ajout d'un script d'installation des dépendances vérifiant automatiquement la présence de Python 3 et du module `venv`.

## 1.0.0
- Initialisation du plugin ACRE SPC pour Jeedom.
